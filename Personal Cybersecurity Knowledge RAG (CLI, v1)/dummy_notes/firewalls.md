# Network Security Firewalls

Firewalls inspect incoming and outgoing traffic. They operate at different layers of the OSI model.

## Stateful Firewalls
Stateful firewalls track the state of active network connections. They block unsolicited incoming traffic unless it belongs to an established connection.

## Stateless Firewalls
Stateless firewalls filter packets based on static rules (e.g. source IP, port, protocol) without tracking connection states. They are faster but less secure than stateful firewalls.

## Next-Gen Firewalls
Next-Generation Firewalls (NGFW) combine traditional firewall capabilities with network device filtering functions, such as deep packet inspection and application awareness.
